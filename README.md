# Книга рецептов
Лабораторная работа номер 1.
Проект построен на Spring Boot и использует Spring Data JPA / Hibernate для работы с данными. В качестве базы данных применяется встроенная H2.

Основной функционал:
- CRUD рецептов;
- поиск по ингредиентам;
- список покупок (генерируется по рецепту).

Основные сущности:
- Recipe - рецепт;
- Ingredient - ингредиент рецепта;
- Step - шаг приготовления;
- Category - категория рецепта.

Для работы проекта нужно иметь Java версии 17 или выше, а также другие необходимые утилиты.
Запуск проекта осуществляется с помощью команды .\mvnw spring-boot:run, будучи в установленной папке.
